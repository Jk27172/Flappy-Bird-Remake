# Flappy Bird Remake

A simple Flappy Bird remake developed using Unity and C#.

## Features
- Bird movement physics
- Pipe spawning system
- Score system
- Game over detection
- Background music and sound effects

## Technologies
- Unity
- C#
- Visual Studio

## Gameplay
Press SPACE to fly and avoid pipes.
